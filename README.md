# Traders Circle

## Overview
Traders Circle is a copy-trading platform that allows traders to connect their MT5 accounts and follow expert traders. Slave accounts can automatically mirror the trades of Master accounts based on their performance. The platform integrates a subscription-based model, performance evaluation, and real-time trading analytics.

## Features
- **Master-Slave Account Model**: Slave accounts copy trades from selected Master accounts.
- **Subscription-Based Access**: Slave accounts subscribe to Master accounts via monthly payments.
- **Performance Evaluation**: Master accounts must maintain an 80% win rate for activation.
- **MT5 Account Integration**: Seamless connection with MT5 trading accounts.
- **Admin Panel**: User management, trade monitoring, and billing oversight.
- **Secure Transactions**: Two-Factor Authentication (2FA) and KYC verification.

## Technology Stack
- **Backend**: Django, PostgreSQL, MT5 API
- **Frontend**: React.js, TailwindCSS
- **Payment System**: Paystack

## Installation

### Prerequisites
